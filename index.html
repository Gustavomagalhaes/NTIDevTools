PRÉ-REQUISITOS

Adicionar a versão do Java como variável do sistema ($JAVA_HOME)

Passo-a-passo:

1. Editar arquivo .bash_profile:
$ vim .bash_profile

2. Adicionar caminho do Java ao arquivo:
JAVA_HOME=/usr/bin/java
export path=$PATH:$JAVA_HOME/bin

3. Salvar e fechar arquivo
Pressionar tecla Esc e digitar :wq

4. Criar pasta Programs:
$ mkdir Programs

INSTRUÇÕES

Passo-a-passo: (SDK)

1. Clicar no botão download.
2. Salvar o arquivo na pasta Downloads.
3. Criar a pasta Programs*. *Caso já tenha criado, pular para o passo 4.
4. Descompactar o arquivo na pasta Programs.
5. Acessar a pasta do programa e executá-lo*.
*Caso necessário, configurar o caminho do JDK. (Menu Window -> Preferences -> Java -> Installed JREs).

Passo-a-passo: (Plugins)

1. Clicar no botão download.
2. Salvar o arquivo na pasta Downloads.
3. Criar a pasta Programs*. *Caso já tenha criado, pular para o passo 4.
4. Instalar plugin no Eclipse
Menu Help -> Install New Software -> Add -> Name: NomeDoPlugin | Location: CaminhoDoPlugin -> OK

Passo-a-passo: (Ferramentas)

1. Clicar no botão download.
2. Salvar o arquivo na pasta Downloads.
3. Criar a pasta Programs*. *Caso já tenha criado, pular para o passo 4.
4. Configurar variável do sistema seguindo os passos nos pré-requisitos.

Shutter

Ferramenta para tirar screenshot da tela.

1. Instalar:
$ sudo yum install shutter

2. Screenshot da tela inteira:
Print Screen SysRq

3. Screenshot de uma área delimitada:
shift + Print Screen SysRq

4. Screenshot de uma janela inteira:
alt + Print Screen SysRq
